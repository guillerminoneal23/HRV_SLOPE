HRV Slope App — Quick Start
============================

Welcome to HRV Slope App, a local recovery-response monitor based on the
RMSSD-Slope method (Naranjo Orellana et al., 2019).


Getting Started
---------------

1. Extract the ENTIRE folder to a location you control.
   Good locations: Desktop, Documents, or a USB drive.

2. Do NOT extract to C:\Program Files or C:\Windows.
   Windows may block write access in those locations.

3. Double-click hrv_slope_app.exe to start the app.

4. No installation is required. No internet connection is needed.


Your Data
---------

- All data is stored locally inside this folder.
  DO NOT delete files inside the folder — they contain your athlete
  and session data.

- Exported CSV files are saved to the 'exports' subfolder next to the .exe.


Moving or Backing Up
--------------------

- To move the app: copy the ENTIRE folder to the new location.

- To back up your data: copy the entire folder, or at minimum the
  database file (*.sqlite) to a safe location.

- To share the app with a colleague: copy the entire folder.
  Each copy maintains its own independent local database.


Requirements
------------

- Windows 10 or later (64-bit / x64)
- No internet connection required
- No Microsoft account required
- No installation required


In-App Help
-----------

The app includes a built-in Instructions tab with detailed guidance on:
- Measurement protocols
- Data entry workflows
- Interpreting recovery-response results
- Using reports, longitudinal trends, and nomograms


Important Notice
----------------

This tool supports training-load monitoring and recovery-response review.
It is NOT a medical diagnostic instrument.

Results should be interpreted with coaching context, workload history,
recovery conditions, sleep, travel, illness, and protocol consistency.


Reference
---------

Naranjo Orellana, J., et al. (2019).
RMSSD-Slope method for internal training load monitoring.
